# Análise de Acidentes de Trânsito no Brasil

##  Objetivo

Este projeto tem como objetivo analisar dados de acidentes de trânsito no Brasil utilizando técnicas de análise exploratória de dados e visualização interativa.

Através de indicadores, gráficos e filtros dinâmicos, busca-se identificar padrões relacionados à gravidade dos acidentes, regiões mais afetadas e fatores associados às ocorrências.

---

##  Tecnologias Utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit
* GitHub

---

##  Funcionalidades

* Filtros interativos
* KPIs dinâmicos
* Análise temporal
* Correlação estatística
* Dashboard interativo
* Visualização da gravidade dos acidentes
* Comparação entre regiões

---

##  Indicadores

* Total de acidentes
* Total de feridos
* Total de mortes
* Média de veículos envolvidos

---

pip install -r requirements.txt
