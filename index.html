🚗 Análise de Acidentes de Trânsito no Brasil

Projeto da disciplina Linguagem de Programação

📌 Objetivo

Este projeto realiza uma análise exploratória dos acidentes de trânsito ocorridos no Brasil.

O objetivo é identificar padrões, regiões mais afetadas e fatores relacionados à gravidade dos acidentes.

🛠 Tecnologias Utilizadas

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Streamlit
  • GitHub

📊 Funcionalidades

  • Filtros interativos
  • KPIs dinâmicos
  • Análise temporal
  • Correlação estatística
  • Dashboard interativo
